# TheGeek

THE GEEK is an online bookstore app that enables users to buy, sell, and read books on a single platform. Built with Java in Android Studio, it integrates Google Books API, Hemberg API, and Firebase Realtime Database for seamless book discovery, e-reading, and secure data management.

The repository includes sample screenshots for preview and demonstration purposes. These represent only a few examples—there are many more pages and features within the application beyond those shown.
